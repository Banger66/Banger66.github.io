CSS - Margin, Border and Padding

Using CSS we can wrap the content, we can set the distance between the content and the border, and the space with other bits and pieces on the page. For that, we need to use a few attributes (special instructions)

  • Margin - This is the bit of space that sits outside of your border. Think of it like no mans land - like a boundary between two borders.
  • Border - Just like a country, this is a line that defines the edge of the element. You can add a border on all sides to an element - or you can choose any single or combination of sides.
  • Padding - The bit of space between the content and the border. Things tend to look a lot nicer if you give your content a bit of breathing room!
  • Content - the content can be whatever you like, for example text, image, video, etc

This is how we apply the above attributes to an HTML tag:

       
  div {
  width: 200px;
  padding: 15px;
  border: black 5px solid;
  margin: 20px; 
  }

Where div is our content, we've given it 15 pixels of padding, a solid black 5 pixel border, and a margin of 20 pixels.
